# @candlefinance/slider

A little (horizontal) animated slider component for React Native.

## Installation

```sh
npm install @candlefinance/slider
```

## Usage

```js
import { multiply } from '@candlefinance/slider';

// ...

const result = await multiply(3, 7);
```
